## Site vitrine — démarrage

```bash
npm install
npm run dev      # aperçu local
npm run build    # génère /dist prêt à héberger
```

Déploiement en un clic sur Vercel ou Netlify. Voir [`DOMAINE_ET_LANCEMENT.md`](./DOMAINE_ET_LANCEMENT.md).
